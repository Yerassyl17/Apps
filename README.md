# Fun Spark

Small entertainment web app with instant jokes, trivia, and a mini game. Works
offline as a static site and includes English/Russian language switching.

## Run

### Option 1: Python (recommended)

```bash
python -m http.server 8000
```

Open `http://localhost:8000` in your browser.

### Option 2: Node.js (if Python is unavailable)

```bash
npx serve .
```

Open the URL printed in the terminal.

### Option 3: Directly open the file

You can also open `index.html` directly in your browser (drag & drop or double
click). Some browsers may restrict certain features when opened from a local
file, so running a local server is preferred.

## Запуск

### Вариант 1: Python (рекомендуется)

```bash
python -m http.server 8000
```

Откройте `http://localhost:8000` в браузере.

### Вариант 2: Node.js (если нет Python)

```bash
npx serve .
```

Откройте URL, который будет показан в терминале.

### Вариант 3: Открыть файл напрямую

Можно открыть `index.html` напрямую в браузере (перетащить файл или
кликнуть дважды). В некоторых браузерах есть ограничения при открытии
локального файла, поэтому локальный сервер предпочтительнее.
